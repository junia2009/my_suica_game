# スライム合成パズル 🟢

Matter.js + Canvas2D で作った、ブラウザ向けの合成パズルゲーム（スイカゲーム風）。
小さなスライムを箱に落とし、**同じ種類が触れると 1 つ上位のスライムに合体**します。
合体を繰り返して最上位「スライムキング」を目指しましょう。

## 遊び方

`index.html` をブラウザで開くだけ。ビルド不要・単一ページで動きます。

| 入力 | 動作 |
|------|------|
| マウス移動 / タッチ移動 | 投下位置を左右に移動 |
| クリック / タップ | スライムを落とす |
| ← → キー | 投下位置を移動 |
| Space / ↓ キー | 落とす |

- 落とせるのは rank 1〜5。上位は合体でのみ出現します。
- 箱の上部のデッドラインを **1.5 秒以上** 超え続けるとゲームオーバー。
- 短時間に連続で合体させると **連鎖ボーナス**（スコア倍率アップ）。
- スライムキング同士がぶつかると消えて **大ボーナス**。

## 特徴

- 物理形状は常に円（Matter.js）。**見た目の変形・表情はすべて描画層で**生成。
- ぷにぷに squash & stretch、ランクごとに変化する顔・アクセサリ（ほっぺ／ツノ／王冠／羽／星柄／マント…）。
- 合体エフェクト（粒＋波紋）、WebAudio による効果音（右下ボタンで ON/OFF）。
- ハイスコアは localStorage に保存。
- モバイル対応（タッチ操作・DPR 対応・スクロール抑制）。

## 技術スタック

- 単一 HTML ファイル + Vanilla JS
- [Matter.js](https://brm.io/matter-js/) 0.19.0（`matter.min.js` として同梱。CDN フォールバックあり）
- Canvas 2D 描画 / WebAudio
